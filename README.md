## dgiot_topo通道

+ dgiot_topo通道是影子设备与前端组态页面之间的桥接通道，
+ 实时订阅dgiot_task通道中解析成物模型的消息，
+ 将物模型消息格式转换成knova组态json格式推送到前端组态页面，实现大屏上设备数据的实时更新
